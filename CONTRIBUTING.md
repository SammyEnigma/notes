# Contributing guidelines

* Be respectful.
* Test your code thoroughly before submitting a pull request.
* If changes were made to the UI, add 'before' and 'after' screenshots.

## AI Policy

* AI tools may be used to assist in writing code, but the final code must be reviewed and approved by a human before being submitted as a pull request. It is crucial that you understand what you are changing.
* PR titles *MUST* be written manually by humans.
* PR descriptions, issues, and comments may be assisted by AI tools, but they *MUST* be reviewed by a human before submission.
